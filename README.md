# 📚 Learning Portal

A full-stack Learning Portal built using the MERN Stack.

## Features

- 🎥 Video Learning
- 📌 Multiple Bookmarks
- ▶ Resume from Bookmark
- 💾 Auto Save Progress
- 🔍 Search Videos
- 📊 Progress Bar
- 🛡 Screenshot Protection
- 🌐 Responsive UI
- ⚡ Toast Notifications

## Tech Stack

Frontend
- React
- Vite
- Tailwind CSS
- Axios
- React Router
- React Hot Toast

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```

## Environment Variables

Create a `.env` inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_uri
```
